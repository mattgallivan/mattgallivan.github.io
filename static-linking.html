Static Libraries in GCC

Here's how to create a static library and link it using gcc. 

	  gcc -c -o math.o math.c
	  ar -rs libs/libmath.a math.o
	  gcc -c -o app.o app.c
	  gcc -o app.o app -Llib -lmath

A few things to note:

  • The location (before or after the object files and executable) of where you the link the library file using -l matters.
  • The name of the library file has to be prefixed with lib when using -l.
